Roleplay setup:
- You need a new software tool for your work and you ask the IT support person to install it on your computer.
- The AI plays the IT support person who helps with software on work computers.
- Goal: Explain what tool you need and why, and agree on when it can be installed.
> Hi, I need a new tool installed on my laptop.
> Can you help me install some software for my work?
> I want to ask about getting a program set up on my computer.
